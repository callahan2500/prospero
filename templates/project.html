Design an educational chatbot

To get started, read sections 1-3. Then briefly write a problem statement and then begin Step 1.

1. Company Overview

We are a leading company specializing in creating chatbots for virtual educational organizations, including bootcamps, online universities, and homeschooling platforms. Our mission is to enhance the educational experience through intuitive and AI-powered chatbot solutions.

2. Objective

Design a customizable chatbot that aligns with each client's branding. This chatbot should cater to educational programs serving people of all ages. The primary goal is to facilitate efficient communication between teachers and students, with features that aid in teaching and learning.

3. Problem Statement

Use the information provided about the company and objective and draft a short problem statement

Step 1: Empathize

In this step, you will learn about the needs, challenges, and expectations of potential users. If you're stuck, click the help button below!

  • Task 1: Create a Google folder named "Educational Chatbot Project" and set up a document for interview notes. Engage with 10 representatives from educational organizations, documenting a concise summary for each interaction.

  • Task 2: Analyze your notes and draft 2-3 user personas. Use this Figma template to structure and present your personas.

  • Task 3: Create an affinity map to categorize and prioritize features. Utilize tools like Miro or FigJam for this exercise.

Step 2: Define

Now you will articulate the specific problems and needs the chatbot should address based on your insights from Step 1. If you're stuck, click the help button below!

  • Task 1: Using the provided template, list down the primary challenges and needs identified from the interviews.

  • Task 2: Use the provided template to conduct competitive research on existing educational chatbots.

Step 3: Ideate

Using the previous two steps, brainstorm potential solutions and design directions for the chatbot. If you're stuck, click the help button below!

  • Task 1: Create a document named "Chatbot Feature Brainstorm" and list all feature ideas and functionalities, based on the insights gathered and the problems identified.

  • Task 2: Based on the brainstorming session, sketch out user flows for the primary chatbot features. Use this Figma template to visualize the user journeys.

  • Task 3:Create low and mid-fidelity wireframes for the chatbot interface using Figma.

  • Task 4:Refine the mid-fidelity wireframes and develop them into high-fidelity wireframes, incorporating detailed UI elements, color schemes, and typography.

Step 4: Prototype

Now you will develop a high fidelity version to test with potential users! If you're stuck, click the help button below!

  • Task 1: Organize user testing sessions with at least 5 participants from educational organizations. Document their feedback in a dedicated Google document.

  • Task 2: Analyze the feedback and compile a list of design changes and improvements.

  • Task 3:Update the Figma prototype with the necessary changes. Once finalized, prepare the chatbot design for handoff to the development team.
